Provider Management
===================

This section covers managing providers. Currently the supported provider types are Amazon Web Services *(AWS)* account and OpenShift Container Platform *(OCP)*. Each provider requires special configuration in order to be created. The specifics for adding each provider type are described below along with information to remove a provider.
